## 🚀 3 PASSOS PARA DEPLOYMENT

### Passo 1: Git Push (5 minutos)
```bash
cd projeto
git add .
git commit -m "Deploy Render + Vercel"
git push origin main
```

### Passo 2: Backend Render (5 minutos)
1. render.com → Conectar GitHub
2. Novo Web Service
3. Repository: `sentinela_ia_v2`
4. Adicionar variáveis de ambiente
5. Deploy! → `https://sentinela-backend.render.com`

### Passo 3: Frontend Vercel (5 minutos)
1. vercel.com → Conectar GitHub
2. Novo projeto
3. Root: `frontend`
4. Adicionar `NEXT_PUBLIC_API_URL`
5. Deploy! → `https://seu-projeto.vercel.app`

**Tempo total: ~15 minutos ⏱️**
